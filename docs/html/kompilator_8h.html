Emits the assembly code after parsing a valid AST.

Simply output a char to stdout. The instruction must be a valid assembly instruction.

kompilator_emit("MOV AX, 2")
Parameters
instructionThe string containing the assembly code.
Note
This will only outputs to stdout the assembly code, it will not save to a file for example.
